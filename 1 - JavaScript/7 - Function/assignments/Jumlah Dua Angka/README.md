# Assignment - Fungsi Menjumlahkan Dua Angka (Easy)

## Tujuan Pembelajaran

- Memahami konsep dasar function di JavaScript.
- Mampu membuat function dengan parameter.
- Mampu mengembalikan nilai menggunakan `return`.
- Mampu menggunakan function untuk melakukan operasi aritmatika sederhana.

## Soal

Buatlah sebuah **function** bernama `sum` yang menerima **dua parameter** berupa angka, kemudian mengembalikan hasil penjumlahannya menggunakan `return`.

> **Catatan:** Fokus utama assignment ini adalah memahami cara membuat function yang menerima parameter dan mengembalikan nilai.

## Result / Test Case

**Input**

```javascript
sum(5, 3);
```

**Output**

```text
8
```
