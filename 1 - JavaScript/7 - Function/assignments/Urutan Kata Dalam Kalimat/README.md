# Assignment - Membalik Urutan Kata dalam Kalimat (Hard)

## Tujuan Pembelajaran

- Memahami function yang memproses data bertipe string.
- Mampu memanipulasi string menggunakan method JavaScript.
- Mampu mengolah data menjadi bentuk yang diinginkan menggunakan function.
- Mampu mengembalikan hasil menggunakan `return`.

## Soal

Buatlah sebuah **function** bernama `reverseSentence` yang menerima **satu parameter** berupa kalimat.

Function harus mengembalikan kalimat tersebut dengan **urutan kata yang dibalik**, tanpa mengubah susunan huruf pada setiap katanya.

Sebagai contoh:

```text
saya belajar javascript
```

menjadi

```text
javascript belajar saya
```

> **Catatan:** Fokus utama assignment ini adalah memahami cara memanipulasi string dengan memisahkan kata, membalik urutannya, kemudian menggabungkannya kembali menjadi sebuah kalimat.

## Result / Test Case

**Input**

```javascript
reverseSentence("saya belajar javascript");
```

**Output**

```text
javascript belajar saya
```
