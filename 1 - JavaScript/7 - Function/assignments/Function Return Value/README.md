# Assignment - Function Return Value (Medium)

## Tujuan Pembelajaran

- Memahami perbedaan antara `return` dan `console.log()`.
- Mampu membuat function yang mengembalikan sebuah nilai.
- Mampu menggunakan nilai hasil `return` untuk proses selanjutnya.
- Mampu menyusun function yang menghasilkan output berdasarkan parameter yang diberikan.

## Soal

Buatlah sebuah **function** bernama `hitungLuasPersegiPanjang` yang menerima dua parameter, yaitu `panjang` dan `lebar`.

Function harus **mengembalikan** hasil perhitungan luas persegi panjang menggunakan keyword `return`, bukan langsung menampilkan hasilnya dengan `console.log()`.

Setelah itu, tampilkan nilai yang dikembalikan oleh function menggunakan `console.log()`.

> **Catatan:** Fokus utama assignment ini adalah memahami penggunaan keyword `return` untuk mengembalikan nilai dari sebuah function agar dapat digunakan kembali pada proses berikutnya.

## Result / Test Case

**Input**

```javascript
console.log(hitungLuasPersegiPanjang(10, 5));
```

**Output**

```text
50
```
