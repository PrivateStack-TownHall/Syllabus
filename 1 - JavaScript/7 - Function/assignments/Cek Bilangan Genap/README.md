# Assignment - Mengecek Bilangan Genap (Easy)

## Tujuan Pembelajaran

- Memahami function yang mengembalikan nilai boolean.
- Mampu menggunakan parameter dalam function.
- Mampu menggunakan operator modulus (`%`) untuk menentukan bilangan genap.
- Mampu mengembalikan hasil menggunakan `return`.

## Soal

Buatlah sebuah **function** bernama `isEven` yang menerima **satu parameter** berupa angka.

Function harus mengembalikan:

- `true` jika angka tersebut merupakan bilangan genap.
- `false` jika angka tersebut bukan bilangan genap.

> **Catatan:** Gunakan operator modulus (`%`) untuk menentukan apakah suatu bilangan termasuk genap.

## Result / Test Case

**Input**

```javascript
isEven(10);
isEven(7);
```

**Output**

```text
true
false
```
