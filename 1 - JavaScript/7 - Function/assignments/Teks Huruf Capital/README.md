# Assignment - Mengubah Teks Menjadi Huruf Kapital (Easy)

## Tujuan Pembelajaran

- Memahami function yang menerima parameter bertipe string.
- Mampu memanipulasi data string menggunakan function.
- Mampu mengembalikan hasil dari sebuah function menggunakan `return`.
- Mampu menggunakan method bawaan JavaScript pada string.

## Soal

Buatlah sebuah **function** bernama `toUpper` yang menerima **satu parameter** berupa string, kemudian mengembalikan string tersebut dalam huruf kapital.

> **Catatan:** Fokus utama assignment ini adalah memahami penggunaan function untuk memproses data bertipe string.

## Result / Test Case

**Input**

```javascript
toUpper("hello");
```

**Output**

```text
HELLO
```
