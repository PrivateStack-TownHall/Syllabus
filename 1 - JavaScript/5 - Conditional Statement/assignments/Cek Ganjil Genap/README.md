# Assignment - Cek Bilangan Ganjil atau Genap (Easy)

## Tujuan Pembelajaran

- Memahami penggunaan struktur percabangan (`if...else`).
- Mampu menentukan apakah sebuah bilangan termasuk ganjil atau genap.
- Mampu menyusun pseudocode dan mengimplementasikannya ke dalam program.
- Mampu menggunakan operator modulus (`%`) untuk menyelesaikan permasalahan.

## Soal

Buatlah **pseudocode** dan **program** untuk menentukan apakah sebuah bilangan merupakan **ganjil** atau **genap**.

Program harus menerima sebuah bilangan sebagai input, kemudian menampilkan hasil sesuai dengan jenis bilangannya.

> **Catatan:** Gunakan operator modulus (`%`) untuk menentukan apakah suatu bilangan habis dibagi dua.

## Result / Test Case

**Input**

```text
7
```

**Output**

```text
Ganjil
```

**Input**

```text
10
```

**Output**

```text
Genap
```
