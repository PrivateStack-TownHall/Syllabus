# Assignment - Positif, Negatif, atau Nol (Easy)

## Tujuan Pembelajaran

- Memahami penggunaan struktur percabangan `if...else if...else`.
- Mampu menentukan kondisi berdasarkan nilai yang diberikan.
- Mampu menyusun logika percabangan sederhana menggunakan beberapa kondisi.
- Mampu mengimplementasikan percabangan dasar dalam program JavaScript.

## Soal

Buatlah sebuah program yang menggunakan struktur **`if...else if...else`** untuk menentukan apakah sebuah bilangan termasuk:

- **Positif**, jika nilainya lebih dari `0`.
- **Negatif**, jika nilainya kurang dari `0`.
- **Nol**, jika nilainya sama dengan `0`.

Program harus menerima sebuah bilangan sebagai input, kemudian menampilkan kategori yang sesuai.

> **Catatan:** Fokus utama assignment ini adalah memahami penggunaan `if...else if...else` untuk menangani lebih dari satu kondisi dalam sebuah program.

## Result / Test Case

**Input**

```text
-5
```

**Output**

```text
Negatif
```

**Input**

```text
10
```

**Output**

```text
Positif
```

**Input**

```text
0
```

**Output**

```text
Nol
```
