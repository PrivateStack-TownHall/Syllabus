# Assignment - Ternary untuk Cek Dewasa (Medium)

## Tujuan Pembelajaran

- Memahami penggunaan operator ternary dalam JavaScript.
- Mampu menggunakan operator ternary untuk mengambil keputusan berdasarkan suatu kondisi.
- Mampu menerapkan operator perbandingan dalam sebuah expression.
- Mampu menyusun expression yang ringkas dan mudah dipahami menggunakan operator ternary.

## Soal

Buatlah sebuah program yang menggunakan **operator ternary (`? :`)** untuk menentukan apakah seseorang sudah **Dewasa** atau **Belum Dewasa** berdasarkan usianya.

Gunakan ketentuan berikut:

- Jika umur **lebih dari atau sama dengan 18**, tampilkan **"Dewasa"**.
- Jika umur **kurang dari 18**, tampilkan **"Belum Dewasa"**.

> **Catatan:** Fokus utama assignment ini adalah memahami cara menggunakan operator ternary sebagai alternatif penulisan `if...else` untuk membuat keputusan berdasarkan suatu kondisi.

## Result / Test Case

**Input**

```text
15
```

**Output**

```text
Belum Dewasa
```

**Input**

```text
20
```

**Output**

```text
Dewasa
```
