# Assignment - Program Promo Harbolnas (Medium)

## Tujuan Pembelajaran

- Memahami penggunaan percabangan bertingkat.
- Mampu menerapkan beberapa kondisi dalam satu program.
- Mampu menyusun pseudocode dan mengimplementasikannya ke dalam program.
- Mampu menghitung diskon berdasarkan kode promo dan total pembelian.

## Soal

Sebuah toko material bernama **Maju Makmur** menyediakan dua jenis kode promo, yaitu **Discount10%** dan **DiscountHarbolnas**.

Buatlah **pseudocode** dan **program** untuk menghitung total pembayaran berdasarkan **kode promo** dan **total pembelian**.

### Ketentuan Promo

#### Promo `Discount10%`

- Total ≥ Rp500.000 → Diskon 10%
- Total Rp300.000 – Rp499.999 → Diskon 5%
- Total < Rp300.000 → Diskon 2,5%

#### Promo `DiscountHarbolnas`

- Total ≥ Rp250.000 → Diskon 15%
- Total < Rp250.000 → Tidak mendapat diskon

> **Catatan:** Program harus menghitung total pembayaran setelah diskon diterapkan.

## Result / Test Case

**Input**

```text
Total Pembelian : Rp750.000
Kode Promo : Discount10%
```

**Output**

```text
Total yang harus dibayarkan: Rp675.000
```
