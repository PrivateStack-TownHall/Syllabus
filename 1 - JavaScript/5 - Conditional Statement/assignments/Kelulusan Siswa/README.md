# Assignment 5 - Struktur Percabangan (Easy)

## Tujuan Pembelajaran

- Memahami konsep percabangan dalam penyusunan algoritma.
- Mampu menyusun pseudocode menggunakan struktur percabangan (`IF...ELSE`).
- Mampu menerapkan logika pengambilan keputusan berdasarkan suatu kondisi.
- Mampu menyelesaikan permasalahan sederhana menggunakan pseudocode.

## Soal

Buatlah sebuah **pseudocode** untuk menentukan apakah seorang siswa dinyatakan **Lulus** atau **Tidak Lulus** berdasarkan nilai yang diperoleh.

Gunakan ketentuan berikut:

- Jika nilai **lebih dari atau sama dengan 75**, maka hasilnya **Lulus**.
- Jika nilai **kurang dari 75**, maka hasilnya **Tidak Lulus**.

> **Catatan:** Fokus utama assignment ini adalah melatih penggunaan struktur percabangan (`IF...ELSE`) dalam menyelesaikan sebuah permasalahan.

## Result / Test Case

**Input**

```text
80
```

**Output**

```text
Lulus
```

**Input**

```text
60
```

**Output**

```text
Tidak Lulus
```
