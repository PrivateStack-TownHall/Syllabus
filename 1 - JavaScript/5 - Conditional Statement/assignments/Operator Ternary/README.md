# Assignment - Operator Ternary (Easy)

## Tujuan Pembelajaran

- Memahami konsep operator ternary dalam JavaScript.
- Memahami penggunaan operator ternary sebagai bentuk sederhana dari percabangan `if...else`.
- Mampu menggunakan operator ternary untuk menentukan hasil berdasarkan suatu kondisi.
- Mampu menuliskan expression yang lebih ringkas menggunakan operator ternary.

## Soal

Buatlah sebuah program yang menggunakan **operator ternary (`? :`)** untuk menentukan apakah sebuah bilangan termasuk **positif** atau **negatif**.

Program harus menerima sebuah angka sebagai input, kemudian menampilkan hasil sesuai dengan kondisi yang diberikan.

> **Catatan:** Fokus utama assignment ini adalah memahami cara kerja operator ternary sebagai alternatif penulisan `if...else` untuk kondisi yang sederhana.

## Result / Test Case

**Input**

```text
-10
```

**Output**

```text
Negatif
```
