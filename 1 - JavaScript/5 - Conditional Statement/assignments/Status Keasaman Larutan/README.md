# Assignment - Status Keasaman Larutan (Easy)

## Tujuan Pembelajaran

- Memahami penggunaan struktur percabangan (`if...else if...else`).
- Mampu menentukan kategori berdasarkan suatu rentang nilai.
- Mampu menyusun pseudocode dan mengimplementasikannya ke dalam program.
- Mampu menyelesaikan permasalahan menggunakan beberapa kondisi.

## Soal

Buatlah **pseudocode** dan **program** untuk menentukan status keasaman suatu larutan berdasarkan nilai **pH**.

Gunakan ketentuan berikut:

- pH **0–6** → **Asam**
- pH **7** → **Netral**
- pH **8–14** → **Basa**

> **Catatan:** Program hanya menerima nilai pH pada rentang 0 hingga 14.

## Result / Test Case

**Input**

```text
5
```

**Output**

```text
Asam
```

**Input**

```text
7
```

**Output**

```text
Netral
```

**Input**

```text
12
```

**Output**

```text
Basa
```
