# Assignment - Jumlah Hari dalam Bulan (Hard)

## Tujuan Pembelajaran

- Memahami penggunaan struktur percabangan bertingkat.
- Mampu menggabungkan beberapa kondisi dalam satu program.
- Mampu menentukan jumlah hari berdasarkan bulan dan tahun.
- Mampu menerapkan logika tahun kabisat dalam penyelesaian masalah.

## Soal

Buatlah sebuah program untuk menentukan **jumlah hari dalam suatu bulan** berdasarkan **nomor bulan** dan **tahun** yang diinput oleh pengguna.

Program harus memperhatikan aturan berikut:

- Bulan dengan **31 hari**.
- Bulan dengan **30 hari**.
- Bulan **Februari** memiliki:
  - **29 hari** jika tahun tersebut merupakan **tahun kabisat**.
  - **28 hari** jika bukan **tahun kabisat**.

Setelah menentukan jumlah hari, tampilkan hasilnya ke layar.

> **Catatan:** Fokus utama assignment ini adalah menggabungkan beberapa struktur percabangan untuk menyelesaikan sebuah studi kasus yang melibatkan lebih dari satu kondisi.
