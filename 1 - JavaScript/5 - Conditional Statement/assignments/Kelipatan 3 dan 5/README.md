# Assignment - Kelipatan 3 dan 5 (Easy)

## Tujuan Pembelajaran

- Memahami penggunaan operator modulus (`%`).
- Mampu menggunakan struktur percabangan.
- Mampu menentukan apakah suatu bilangan memenuhi lebih dari satu kondisi.
- Mampu menyusun pseudocode dan mengimplementasikannya ke dalam program.

## Soal

Buatlah **pseudocode** dan **program** untuk menentukan apakah sebuah bilangan merupakan **kelipatan 3 sekaligus kelipatan 5**.

Program hanya akan menghasilkan **true** jika bilangan habis dibagi **3** dan **5** sekaligus.

> **Catatan:** Gunakan operator logika (`&&`) untuk menggabungkan kedua kondisi.

## Result / Test Case

**Input**

```text
15
```

**Output**

```text
true
```

**Input**

```text
8
```

**Output**

```text
false
```

**Input**

```text
3
```

**Output**

```text
false
```
