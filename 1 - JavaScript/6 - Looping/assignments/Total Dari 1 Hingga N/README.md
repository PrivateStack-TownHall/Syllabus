# Assignment 7 - Total dari 1 hingga N (Medium)

## Tujuan Pembelajaran

- Memahami konsep perulangan dalam penyusunan algoritma.
- Mampu menyusun pseudocode menggunakan struktur perulangan (`FOR` atau `WHILE`).
- Mampu menghitung total penjumlahan secara bertahap menggunakan proses iterasi.
- Mampu menyelesaikan permasalahan yang melibatkan perulangan dan akumulasi nilai.

## Soal

Buatlah sebuah **pseudocode** untuk menghitung total penjumlahan angka dari **1 hingga N**, di mana **N** merupakan angka yang diinput oleh pengguna.

Program harus menggunakan **struktur perulangan** untuk menjumlahkan setiap angka mulai dari **1** hingga **N**, kemudian menampilkan hasil akhirnya.

> **Catatan:** Fokus utama assignment ini adalah memahami cara kerja perulangan dan proses akumulasi nilai. Hindari menggunakan rumus matematika secara langsung, dan selesaikan permasalahan menggunakan proses iterasi.

## Result / Test Case

**Input**

```text
5
```

**Output**

```text
15
```

**Penjelasan**

```text
1 + 2 + 3 + 4 + 5 = 15
```
