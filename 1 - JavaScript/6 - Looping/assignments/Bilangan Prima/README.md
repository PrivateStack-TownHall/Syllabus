# Assignment - Bilangan Prima (Medium)

## Tujuan Pembelajaran

- Memahami konsep bilangan prima.
- Mampu menggunakan perulangan untuk melakukan pengecekan.
- Mampu menggabungkan perulangan dengan percabangan.
- Mampu menentukan apakah suatu bilangan termasuk bilangan prima.

## Soal

Buatlah sebuah program untuk menentukan apakah sebuah bilangan merupakan **bilangan prima**.

Bilangan prima adalah bilangan yang hanya memiliki **dua faktor**, yaitu **1** dan bilangan itu sendiri.

> **Catatan:** Gunakan proses pencarian faktor untuk menentukan apakah suatu bilangan termasuk bilangan prima.

## Result / Test Case

**Input**

```text
7
```

**Output**

```text
7 adalah Bilangan Prima
```
