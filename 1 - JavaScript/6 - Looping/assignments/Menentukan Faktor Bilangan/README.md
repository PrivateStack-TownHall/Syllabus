# Assignment - Menentukan Faktor Bilangan (Medium)

## Tujuan Pembelajaran

- Memahami penggunaan perulangan dalam menyelesaikan permasalahan matematika.
- Mampu menentukan faktor dari sebuah bilangan.
- Mampu menggunakan operator modulus (`%`) untuk melakukan pengecekan.
- Mampu mengimplementasikan algoritma sederhana menggunakan perulangan.

## Soal

Buatlah sebuah program untuk menampilkan **seluruh faktor** dari sebuah bilangan.

Faktor adalah bilangan yang dapat membagi suatu bilangan **tanpa menghasilkan sisa**.

Sebagai contoh, faktor dari **10** adalah:

- 1
- 2
- 5
- 10

> **Catatan:** Gunakan perulangan untuk memeriksa setiap bilangan mulai dari `1` hingga nilai input.

## Result / Test Case

**Input**

```text
10
```

**Output**

```text
1
2
5
10
```
