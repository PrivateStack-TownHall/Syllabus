# Assignment - Menampilkan Pola Segitiga (Medium)

## Tujuan Pembelajaran

- Memahami penggunaan nested loop (perulangan bersarang).
- Mampu menyusun pola menggunakan perulangan.
- Mampu menggabungkan beberapa perulangan dalam satu program.
- Mampu menyelesaikan studi kasus sederhana menggunakan nested loop.

## Soal

Buatlah sebuah program yang menampilkan pola segitiga menggunakan karakter `#` dan `$`.

Gunakan nilai input sebagai jumlah baris yang akan ditampilkan.

> **Catatan:** Fokus utama assignment ini adalah memahami penggunaan **nested loop** untuk membentuk sebuah pola.

## Result / Test Case

**Input**

```text
5
```

**Output**

```text
#
#$
#$#
#$#$
#$#$#
```
