# Assignment - Mencetak Angka Ganjil Dari 1–10 Menggunakan `for` (Easy)

## Tujuan Pembelajaran

- Memahami penggunaan perulangan `for`.
- Mampu mengulang suatu proses sebanyak jumlah iterasi yang ditentukan.
- Mampu menggunakan variabel sebagai penghitung (counter) dalam perulangan.
- Mampu menampilkan hasil perulangan menggunakan `console.log()`.

## Soal

Buatlah sebuah program yang menggunakan **perulangan `for`** untuk menampilkan angka ganjil **1 hingga 10** secara berurutan.

> **Catatan:** Fokus utama assignment ini adalah memahami struktur dasar perulangan `for`, yang terdiri dari proses inisialisasi, kondisi, dan increment.

## Result / Test Case

**Output**

```text
1
3
5
7
9
```
