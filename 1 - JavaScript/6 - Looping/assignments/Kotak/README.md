# Assignment - Menampilkan Pola Kotak (Hard)

## Tujuan Pembelajaran

- Memahami penggunaan nested loop (perulangan bersarang).
- Mampu menyusun pola dua dimensi menggunakan perulangan.
- Mampu menggabungkan percabangan dan nested loop.
- Mampu menyelesaikan studi kasus pola dengan logika yang lebih kompleks.

## Soal

Buatlah sebuah program untuk menampilkan pola **kotak kosong** menggunakan karakter `#`.

Gunakan nilai input sebagai panjang dan lebar kotak.

> **Catatan:** Gunakan **nested loop** dan percabangan untuk menentukan apakah karakter yang dicetak berada pada bagian tepi atau bagian dalam kotak.

## Result / Test Case

**Input**

```text
5
```

**Output**

```text
#####
#   #
#   #
#   #
#####
```
