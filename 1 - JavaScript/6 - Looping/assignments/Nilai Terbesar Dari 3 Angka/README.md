# Assignment 6 - Nilai Terbesar dari 3 Angka (Medium)

## Tujuan Pembelajaran

- Memahami penerapan struktur percabangan dalam algoritma.
- Mampu menyusun pseudocode untuk membandingkan beberapa nilai.
- Mampu menentukan nilai terbesar dari beberapa input menggunakan logika yang sistematis.
- Mampu menyelesaikan permasalahan dengan menggabungkan beberapa kondisi percabangan.

## Soal

Buatlah sebuah **pseudocode** untuk menentukan **nilai terbesar** dari tiga buah angka yang diinput oleh pengguna.

Program harus membandingkan ketiga angka tersebut, kemudian menampilkan angka yang memiliki nilai paling besar.

> **Catatan:** Assignment ini membutuhkan beberapa proses perbandingan menggunakan struktur percabangan. Sebelum menulis pseudocode, cobalah menyusun alur logikanya terlebih dahulu agar solusi yang dihasilkan lebih mudah dipahami.

## Result / Test Case

**Input**

```text
5, 9, 3
```

**Output**

```text
9 adalah nilai terbesar
```
