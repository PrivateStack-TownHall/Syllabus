# Assignment - Menghapus Nilai Duplikat dalam Array (Medium)

## Tujuan Pembelajaran

- Memahami proses pencarian data pada array.
- Mampu membuat array baru berdasarkan kondisi tertentu.
- Mampu menghilangkan data duplikat menggunakan logika pemrograman.
- Mampu mengolah array tanpa menggunakan fungsi bawaan tertentu.

## Soal

Diberikan sebuah array sebagai berikut.

```javascript
const items = [2, 4, 4, 7, 2, 9, 9, 1];
```

Buatlah sebuah program untuk menghasilkan **array baru tanpa nilai duplikat**.

**Ketentuan:**

- Tidak diperbolehkan menggunakan `new Set()`.

> **Catatan:** Fokus utama assignment ini adalah memahami proses pencarian dan pengecekan data pada array menggunakan logika pemrograman.

## Result / Test Case

**Input**

```javascript
const items = [2, 4, 4, 7, 2, 9, 9, 1];
```

**Output**

```text
[2, 4, 7, 9, 1]
```
