# Assignment - Mencari Nilai Terbesar dalam Array (Easy)

## Tujuan Pembelajaran

- Memahami cara mengakses elemen dalam array.
- Mampu mencari nilai terbesar menggunakan logika pemrograman.
- Mampu menggunakan perulangan untuk memproses array.
- Mampu menyelesaikan permasalahan tanpa menggunakan fungsi bawaan.

## Soal

Diberikan sebuah array sebagai berikut.

```javascript
const values = [12, 5, 20, 7, 3];
```

Buatlah sebuah program untuk mencari **nilai terbesar** pada array tersebut.

**Ketentuan:**

- Tidak diperbolehkan menggunakan `Math.max()`.

> **Catatan:** Fokus utama assignment ini adalah melatih logika pencarian nilai terbesar menggunakan perulangan.

## Result / Test Case

**Input**

```javascript
const values = [12, 5, 20, 7, 3];
```

**Output**

```text
20
```
