# Assignment - Menampilkan Angka Genap dalam Array (Easy)

## Tujuan Pembelajaran

- Memahami cara mengakses elemen array menggunakan perulangan.
- Mampu menggunakan percabangan untuk melakukan proses penyaringan data.
- Mampu menentukan bilangan genap menggunakan operator modulus (`%`).
- Mampu menampilkan elemen array berdasarkan kondisi tertentu.

## Soal

Diberikan sebuah array sebagai berikut.

```javascript
const data = [10, 13, 8, 7, 6, 20];
```

Buatlah sebuah program untuk menampilkan **seluruh bilangan genap** yang terdapat di dalam array.

> **Catatan:** Gunakan operator modulus (`%`) untuk menentukan apakah suatu elemen merupakan bilangan genap.

## Result / Test Case

**Input**

```javascript
const data = [10, 13, 8, 7, 6, 20];
```

**Output**

```text
10
8
6
20
```
