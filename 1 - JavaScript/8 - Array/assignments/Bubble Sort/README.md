# Assignment 8 - Bubble Sort (Hard)

## Tujuan Pembelajaran

- Memahami konsep algoritma Bubble Sort.
- Mampu menyusun pseudocode menggunakan percabangan dan perulangan.
- Mampu mengimplementasikan logika pertukaran (swap) untuk mengurutkan data.
- Mampu menyelesaikan permasalahan pengurutan data menggunakan pendekatan algoritmik.

## Soal

Buatlah sebuah **pseudocode** untuk mengurutkan sebuah array angka dari nilai **terkecil ke terbesar** menggunakan algoritma **Bubble Sort**.

Assignment ini memiliki tingkat kesulitan yang lebih tinggi dibandingkan soal-soal sebelumnya. Sebelum mulai menulis pseudocode, cobalah memahami cara kerja algoritma Bubble Sort dengan memecah permasalahan menjadi beberapa langkah sederhana. Setelah setiap langkah dipahami, gabungkan seluruh langkah tersebut menjadi sebuah pseudocode yang utuh.

> **Catatan:** Fokus utama assignment ini adalah proses berpikir dan penyusunan algoritma, bukan penggunaan fungsi atau method bawaan untuk melakukan pengurutan.

[5, 2, 9, 1, 5]
