# Assignment - Selisih Nilai Terbesar dan Terkecil (Medium)

## Tujuan Pembelajaran

- Memahami cara melakukan iterasi pada array menggunakan perulangan.
- Mampu mencari nilai terbesar dan nilai terkecil tanpa menggunakan fungsi bawaan.
- Mampu mengolah data dalam array menggunakan logika pemrograman.
- Mampu menyelesaikan permasalahan yang melibatkan beberapa proses dalam satu program.

## Soal

Diberikan sebuah array yang berisi beberapa angka.

Buatlah sebuah program untuk mencari **nilai terbesar** dan **nilai terkecil** pada array tersebut menggunakan **perulangan**, kemudian hitung **selisih** antara kedua nilai tersebut.

**Ketentuan:**

- Tidak diperbolehkan menggunakan `Math.max()`.
- Tidak diperbolehkan menggunakan `Math.min()`.

> **Catatan:** Fokus utama assignment ini adalah memahami proses pencarian nilai maksimum dan minimum menggunakan perulangan, kemudian mengolah hasilnya untuk mendapatkan selisih kedua nilai tersebut.

## Result / Test Case

**Input**

```javascript
const numbers = [12, 45, 3, 67, 25, 18];
```

**Output**

```text
Nilai Terbesar : 67
Nilai Terkecil : 3
Selisih : 64
```
