# Assignment - Menjumlahkan Seluruh Elemen Array (Easy)

## Tujuan Pembelajaran

- Memahami konsep dasar array di JavaScript.
- Mampu mengakses setiap elemen array menggunakan perulangan.
- Mampu menghitung total nilai dari seluruh elemen array.
- Mampu menyelesaikan permasalahan sederhana menggunakan array.

## Soal

Diberikan sebuah array sebagai berikut.

```javascript
const numbers = [4, 7, 2, 9, 1];
```

Buatlah sebuah program untuk menghitung **jumlah seluruh elemen** pada array tersebut.

> **Catatan:** Fokus utama assignment ini adalah memahami cara mengakses setiap elemen array menggunakan perulangan untuk melakukan proses akumulasi nilai.

## Result / Test Case

**Input**

```javascript
const numbers = [4, 7, 2, 9, 1];
```

**Output**

```text
23
```
