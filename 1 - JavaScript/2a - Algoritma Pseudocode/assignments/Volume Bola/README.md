# Assignment: Volume Bola

## Tujuan Pembelajaran

- Memahami konsep penyusunan algoritma.
- Mampu menuliskan pseudocode berdasarkan sebuah permasalahan.
- Mampu menerapkan rumus volume bola ke dalam algoritma dan pseudocode.

## Soal

Buatlah algoritma dan pseudocode untuk menghitung volume sebuah bola menggunakan rumus berikut.

```
V = 4/3 × π × r × r × r
```

Gunakan nilai π = **3.14**.

## Result / Test Case

**Input**

```
r = 7
```

**Output**

```
Volume Bola = 1436.03
```
