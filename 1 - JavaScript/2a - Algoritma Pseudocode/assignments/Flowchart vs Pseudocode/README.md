# Assignment - Flowchart vs Pseudocode (Easy)

## Tujuan Pembelajaran

- Memahami perbedaan antara flowchart dan pseudocode.
- Memahami fungsi flowchart dan pseudocode dalam perancangan algoritma.
- Mampu menjelaskan suatu algoritma dalam bentuk alur yang sistematis.
- Mampu mengubah pseudocode menjadi langkah-langkah flowchart.

## Soal

Jelaskan perbedaan antara **flowchart** dan **pseudocode** menggunakan bahasamu sendiri.

Selanjutnya, ubahlah pseudocode yang telah kamu buat pada **Assignment 2 - Cek Ganjil Genap** menjadi **alur flowchart** dalam bentuk daftar langkah-langkah secara berurutan. Kamu **tidak perlu menggambar simbol flowchart**, cukup jelaskan alurnya secara sistematis.

> **Catatan:** Fokus utama assignment ini adalah memahami perbedaan antara flowchart dan pseudocode serta bagaimana keduanya dapat digunakan untuk menggambarkan solusi dari sebuah permasalahan.

## Result / Test Case

**Output yang Diharapkan**

- Penjelasan mengenai perbedaan flowchart dan pseudocode menggunakan bahasa sendiri.
- Alur flowchart dari Assignment 2 yang dituliskan dalam bentuk langkah-langkah secara berurutan.
