# Assignment - Penjumlahan Dua Angka (Easy)

## Tujuan Pembelajaran

- Memahami konsep dasar algoritma dan pseudocode.
- Mampu menyusun pseudocode untuk menerima input dari pengguna.
- Mampu melakukan operasi aritmatika sederhana menggunakan algoritma.
- Mampu menyusun langkah-langkah penyelesaian masalah secara sistematis.

## Soal

Buatlah sebuah **pseudocode** untuk menghitung hasil **penjumlahan dua buah angka** yang diinput oleh pengguna.

Program harus menerima dua buah angka sebagai masukan, kemudian menghitung dan menampilkan hasil penjumlahannya.

> **Catatan:** Fokus utama assignment ini adalah melatih kemampuan menyusun algoritma sederhana yang melibatkan proses input, perhitungan, dan output.

## Result / Test Case

**Input**

```text
5, 10
```

**Output**

```text
15
```
