# Assignment: Keliling Lingkaran

## Tujuan Pembelajaran

- Memahami konsep penyusunan algoritma.
- Mampu menuliskan pseudocode berdasarkan sebuah permasalahan.
- Mampu menerapkan rumus keliling lingkaran ke dalam algoritma dan pseudocode.

## Soal

Buatlah algoritma dan pseudocode untuk menghitung keliling sebuah lingkaran menggunakan rumus berikut.

```
K = 2 × π × r
```

Gunakan nilai π = **3.14**.

## Result / Test Case

**Input**

```
r = 7
```

**Output**

```
Keliling Lingkaran = 43.96
```
