# Assignment: Volume Tabung

## Tujuan Pembelajaran

- Memahami konsep penyusunan algoritma.
- Mampu menuliskan pseudocode berdasarkan sebuah permasalahan.
- Mampu menerapkan rumus volume tabung ke dalam algoritma dan pseudocode.

## Soal

Buatlah algoritma dan pseudocode untuk menghitung volume sebuah tabung menggunakan rumus berikut.

```
V = π × r × r × t
```

Gunakan nilai π = **3.14**.

## Result / Test Case

**Input**

```
r = 7
t = 10
```

**Output**

```
Volume Tabung = 1538.6
```
