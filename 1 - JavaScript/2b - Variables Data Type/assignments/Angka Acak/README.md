# Assignment 3 - Angka Acak (Medium)

## Tujuan Pembelajaran

- Memahami penggunaan fungsi bawaan JavaScript.
- Memahami tipe data numerik.
- Mampu menghasilkan nilai acak dalam rentang tertentu.
- Mampu menerapkan logika sederhana dalam pemrograman.

## Soal

Buatlah sebuah program untuk menghasilkan **angka acak dari 1 hingga 10**.

Setiap kali program dijalankan, angka yang dihasilkan dapat berbeda.

> **Catatan:** Gunakan fungsi bawaan JavaScript untuk menghasilkan angka acak, kemudian sesuaikan hasilnya agar berada pada rentang 1 hingga 10.

## Result / Test Case

**Output (Contoh)**

```text
5
```

atau

```text
2
```

atau angka lain antara **1 sampai 10**.
