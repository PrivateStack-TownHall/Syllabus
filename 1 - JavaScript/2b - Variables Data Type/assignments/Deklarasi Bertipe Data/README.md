# Assignment - Deklarasi Variabel dengan Tipe Data (Easy)

## Tujuan Pembelajaran

- Memahami penggunaan variabel dalam JavaScript.
- Memahami tipe data primitif seperti `string`, `number`, dan `boolean`.
- Mampu mendeklarasikan variabel dengan tipe data yang sesuai.
- Mampu menyimpan dan menampilkan data menggunakan variabel.

## Soal

Buatlah tiga buah variabel untuk merepresentasikan sebuah produk dengan ketentuan sebagai berikut:

- `nama` bertipe data **string**
- `harga` bertipe data **number**
- `tersedia` bertipe data **boolean**

Kemudian tampilkan nilai dari ketiga variabel tersebut.

> **Catatan:** Fokus utama assignment ini adalah memahami cara mendeklarasikan variabel dan memilih tipe data yang sesuai untuk setiap nilai.

## Result / Test Case

**Output**

```text
Keyboard Mechanical
750000
true
```
