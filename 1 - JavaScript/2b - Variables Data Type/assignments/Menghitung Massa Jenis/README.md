# Assignment - Menghitung Massa Jenis (Easy)

## Tujuan Pembelajaran

- Memahami penggunaan variabel untuk menyimpan data.
- Memahami penggunaan tipe data numerik dalam pemrograman.
- Mampu menyusun algoritma, pseudocode, dan program sederhana berdasarkan rumus matematika.
- Mampu menerapkan operasi aritmatika menggunakan variabel.

## Soal

Buatlah **algoritma**, **pseudocode**, dan **program** untuk menghitung **massa jenis** menggunakan rumus berikut.

```text
ρ = massa / volume
```

Program harus menerima nilai **massa** dan **volume** sebagai input, kemudian menampilkan hasil perhitungan massa jenis.

> **Catatan:** Fokus utama assignment ini adalah memahami penggunaan variabel untuk menyimpan nilai dan melakukan operasi aritmatika sederhana.

## Result / Test Case

**Input**

```text
Massa = 100
Volume = 20
```

**Output**

```text
Massa Jenis = 5
```
