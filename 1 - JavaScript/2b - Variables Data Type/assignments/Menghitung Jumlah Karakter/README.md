# Assignment - Menghitung Jumlah Karakter (Easy)

## Tujuan Pembelajaran

- Memahami penggunaan tipe data string.
- Memahami properti pada string.
- Mampu menghitung jumlah karakter dalam sebuah string.
- Mampu menggunakan variabel untuk menyimpan data.

## Soal

Buatlah sebuah program untuk menghitung **jumlah karakter** dalam sebuah string.

Program harus menerima sebuah string sebagai input, kemudian menampilkan jumlah karakter yang terdapat pada string tersebut.

> **Catatan:** Perhitungan jumlah karakter termasuk spasi, kecuali jika ditentukan sebaliknya.

## Result / Test Case

**Input**

```text
Selamat pagi
```

**Output**

```text
12
```
