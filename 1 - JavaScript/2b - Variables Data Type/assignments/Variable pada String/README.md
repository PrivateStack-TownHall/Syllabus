# Assignment - Variabel pada String (Easy)

## Tujuan Pembelajaran

- Memahami konsep variabel dalam pemrograman.
- Memahami penggunaan tipe data string.
- Mampu menggabungkan variabel dengan teks (string concatenation atau template literal).
- Mampu menampilkan data menggunakan variabel.

## Soal

Buatlah sebuah program yang menampilkan kalimat berikut.

```text
Selamat pagi, Admin
```

Namun, kata **"Admin"** tidak boleh ditulis secara langsung. Simpan terlebih dahulu ke dalam sebuah variabel, kemudian tampilkan hasilnya menggunakan variabel tersebut.

> **Catatan:** Fokus utama assignment ini adalah memahami bagaimana variabel dapat digunakan untuk menyimpan dan menampilkan data bertipe string.

## Result / Test Case

**Output**

```text
Selamat pagi, Admin
```
