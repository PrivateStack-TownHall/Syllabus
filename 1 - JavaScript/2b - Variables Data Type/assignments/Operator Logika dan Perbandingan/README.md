# Assignment - Operator Logika dan Perbandingan (Medium)

## Tujuan Pembelajaran

- Memahami penggunaan operator perbandingan.
- Memahami penggunaan operator logika.
- Memahami perbedaan antara operator `==` dan `===`.
- Mampu menganalisis hasil evaluasi sebuah ekspresi dalam JavaScript.

## Soal

Perhatikan potongan kode berikut.

```javascript
let x = 1;
let y = "1";

let z = x == y;
let p = y + x;
let q = x === y;
let r = y - x;
let total = undefined == false || true;
```

Tentukan hasil dari variabel berikut:

- `z`
- `p`
- `q`
- `r`
- `total`

Kemudian jelaskan **alasan** mengapa setiap variabel menghasilkan nilai tersebut.

> **Catatan:** Fokus utama assignment ini adalah memahami cara kerja operator perbandingan, operator logika, serta proses konversi tipe data (type coercion) dalam JavaScript.

## Result / Test Case

**Output yang Diharapkan**

- Nilai dari setiap variabel (`z`, `p`, `q`, `r`, dan `total`).
- Penjelasan mengenai alasan mengapa setiap ekspresi menghasilkan nilai tersebut.
