# Assignment - Operator Aritmatika (Easy)

## Tujuan Pembelajaran

- Memahami penggunaan operator aritmatika dalam JavaScript.
- Mampu melakukan operasi perhitungan dasar menggunakan operator aritmatika.
- Memahami fungsi masing-masing operator aritmatika.
- Mampu menampilkan hasil perhitungan menggunakan `console.log()`.

## Soal

Buatlah sebuah program yang melakukan operasi aritmatika terhadap dua buah angka, meliputi:

- Penjumlahan (`+`)
- Pengurangan (`-`)
- Perkalian (`*`)
- Pembagian (`/`)
- Modulus (`%`)

Gunakan dua buah variabel untuk menyimpan nilai angka, kemudian tampilkan hasil dari setiap operasi menggunakan `console.log()`.

> **Catatan:** Fokus utama assignment ini adalah memahami fungsi masing-masing operator aritmatika dan cara menggunakannya dalam sebuah program.

## Result / Test Case

**Input**

```text
10 dan 3
```

**Output**

```text
13
7
30
3.3333333333333335
1
```
