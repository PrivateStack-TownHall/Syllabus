# Assignment - Operator Logika (Easy)

## Tujuan Pembelajaran

- Memahami penggunaan operator logika dalam JavaScript.
- Memahami fungsi operator `&&` (AND) dan `||` (OR).
- Mampu menggabungkan beberapa kondisi dalam sebuah ekspresi logika.
- Mampu menganalisis hasil evaluasi dari operator logika.

## Soal

Buatlah sebuah program yang menunjukkan penggunaan operator logika `&&` (AND) dan `||` (OR) untuk mengecek dua kondisi sekaligus.

Gunakan minimal dua kondisi yang berbeda, kemudian tampilkan hasil evaluasinya menggunakan `console.log()`.

Setelah itu, jelaskan mengapa setiap ekspresi menghasilkan nilai `true` atau `false`.

> **Catatan:** Fokus utama assignment ini adalah memahami cara kerja operator logika dalam menggabungkan beberapa kondisi pada JavaScript.

## Result / Test Case

**Contoh Input**

```javascript
console.log(10 > 5 && 8 > 3);
console.log(10 > 5 || 8 < 3);
```

**Contoh Output**

```text
true
true
```
