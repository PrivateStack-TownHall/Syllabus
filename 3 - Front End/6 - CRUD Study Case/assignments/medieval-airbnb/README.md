# 🏰 Medieval AirBnB — Full Stack Marketplace Challenge

**Hard Level — Full Stack (React, Express, PostgreSQL, Firebase)**

<img src="../../../../.gitbook/assets/MA banner.png" alt="" width="50%">

## 📖 Story

Di era abad pertengahan, para **Lords** menyewakan kastil megah, pondok kayu, menara penyihir, hingga arena jousting.\
Para **Travelers** keliling kerajaan untuk mencari tempat menginap yang aman dan nyaman sebelum perjalanan panjang.

Kamu diminta membangun aplikasi marketplace seperti **Airbnb** namun bertema _Medieval Fantasy_ — lengkap dengan cart system, checkout, invoice, multi-role, notifikasi, dan relasi database yang kompleks.

## 🧙 Roles

* **Admin** – mengatur seluruh data.
* **Lord / Host** – membuat dan mengelola listing.
* **Traveler / Guest** – browsing, add-to-cart, checkout, invoice.

## 🛠️ Tech Stack

* **Frontend:** React + Tailwind + Firebase Auth
* **Backend:** Express.js + PostgreSQL (Sequelize ORM)
* **Deployment:** Render + Firebase Hosting

Aplikasi harus mendukung:

* 10+ tabel
* Relasi 1:1, 1:m, m:m
* Cart → Checkout → Invoice flow
* Availability logic
* Notifications
